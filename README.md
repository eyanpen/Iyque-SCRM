# 护城河 SCRM

**面向价值投资顾问的一体化客户关系与咨询管理平台**

基于企业微信 · AI 会话洞察 · 大师方法论沉淀 · 私有化部署

[English](./README.en.md) | 简体中文

---

## 💡 产品定位

护城河 SCRM 服务于家族办公室、私人银行、私募机构、独立投资顾问工作室。
我们将本杰明·格雷厄姆、沃伦·巴菲特、查理·芒格、彼得·林奇、段永平、林园、
但斌、章盟主等价值投资大师的方法论沉淀为可复用的投顾流程，让每一次客户
对话都能传递专业。

核心痛点回应：

- 高净值客户咨询高度个性化，SaaS 通用 SCRM 无法承载投研深度。
- 投顾专家经验隐性化，团队新人难以复用大师方法论。
- 会话合规存档、敏感词拦截、监管报送需求日益严格。
- 数据主权与私有化部署要求突出。

## 🏗️ 系统能力

| 链路 | 能力 |
|------|------|
| **拓客** | 员工活码 / 短链 / 群活码 / H5 落地页，全域承接高净值客户咨询 |
| **档案** | 客户传承档案 + 投顾专家画像 + 标签体系（风险偏好、持仓、传承需求） |
| **咨询** | 会话存档 + AI 语义分析 + 大师方法论 RAG 引用 |
| **合规** | 敏感词拦截规则 + 违规会话预审 + 完整操作日志 |
| **知识** | 投研知识库（大师原著 / 经典案例 / 中国资本市场事件） |
| **AI**   | AI 投顾助手 + AI 会话总结 + AI 意向客户识别 + AI 热词洞察 |

## ✨ 核心亮点

| 亮点 | 说明 |
|------|------|
| 🎓 大师方法论 | 内置格雷厄姆《证券分析》、巴菲特致股东信、芒格《穷查理宝典》、彼得·林奇《One Up on Wall Street》、段永平雪球问答等作为知识库素材 |
| 🇨🇳 中国资本市场语料 | 覆盖茅台、腾讯、网易、拼多多、比亚迪、宁德时代等价投经典案例 |
| 🤖 AI 驱动 | 基于 LangChain4j + RAG，秒级引用大师原话与经典案例 |
| 🔒 合规存档 | 会话存档、敏感词拦截、操作日志一站式满足监管 |
| 🏢 私有化部署 | 数据完全在客户内网，无云服务依赖 |
